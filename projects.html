Android Application Projects:

(1.) Notepad

Notepad is the android application in which we can perform below tasks:-

  • Create user account by email password
  • Login to user account by email password
  • Create new notes
  • Update existing notes
  • Delete any of existing notes

(2.) Music Player

Music Player is the offline mp3 songs player for android mobiles:-

  • Disply all songs
  • Can play any songs from them
  • Pause, Play, Previous buttons to switch songs
  • Search songs from them
  • Shuffle and Repeat option

(3.) Contact App

Contact App is the android application that display contacts and contain some more features:-

  • Disply all phone contacts
  • Can call any number
  • Send text sms to number
  • Search contacts by numbers or name
  • Add new contact
  • Delete any contact
